ConnectGen - Intergenerational Platform

Full-stack web application • School project

Overview

Built a web platform that connects youth and seniors in Singapore through digital storytelling and messaging. The goal was to address social isolation among seniors while facilitating meaningful intergenerational connections.

What I Built

  • User authentication system with secure password storage using bcrypt
  • Private messaging feature allowing one-on-one conversations
  • Discussion forum with posts, comments, and category filtering
  • Accessible interface designed with larger fonts and simplified navigation for seniors
  • Database schema managing users, messages, posts, and relationships

Technical Challenges

This was my first full-stack project. The main challenges were implementing secure authentication, designing an accessible UI for seniors, and managing the database relationships between users, messages, and forum posts.
